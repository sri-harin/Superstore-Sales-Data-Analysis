# 📊 Superstore Sales Data Analysis

## 📌 Project Overview

This project analyzes the Sample Superstore dataset using Python, Pandas, and Matplotlib. The objective is to explore sales data, perform exploratory data analysis (EDA), identify business trends, and visualize key insights using different charts.

---

## 🎯 Project Objective

- Analyze sales and profit performance.
- Perform data cleaning and exploration.
- Identify category-wise and region-wise business trends.
- Visualize insights using charts.
- Build a complete end-to-end data analytics project.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset Information

- Dataset: Sample Superstore
- Total Records: 9,994
- Total Columns: 13

## 🔍 Key Business Insights

- Technology category generated the highest total sales.
- Technology category achieved the highest total profit.
- GroupBy operations simplified category-wise analysis.
- Sorting identified the highest and lowest performing records.
- Charts helped visualize business performance effectively.

3. Install the required libraries:

```bash

```

4. Open `Superstore_Sales_Data_Analysis.ipynb`.
5. Run all cells to reproduce the analysis and charts.
